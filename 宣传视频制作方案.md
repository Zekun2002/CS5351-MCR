# RuoYi-Vue 宣传视频制作方案

## 项目概述
RuoYi-Vue是一套基于SpringBoot+Vue前后端分离的Java快速开发框架，版本v3.8.9，完全开源免费。

## 视频制作建议

### 1. 视频基本信息
- **时长**: 2-3分钟
- **分辨率**: 1920x1080 (Full HD)
- **帧率**: 30fps
- **格式**: MP4
- **音频**: 背景音乐 + 旁白解说

### 3. 视觉设计建议

#### 色彩方案
- 主色调: #409EFF (Element UI 蓝色)
- 辅助色: #67C23A (成功绿)、#E6A23C (警告橙)
- 背景色: 渐变蓝色或白色简洁背景

#### 动画效果
- 文字出现: 淡入 + 轻微上移
- 界面切换: 滑动过渡
- 图标展示: 缩放 + 旋转
- 按钮效果: 悬停放大

#### 字体建议
- 中文: 思源黑体 / 微软雅黑
- 英文: Roboto / Arial
- 代码: Consolas / Monaco

### 4. 音频建议

#### 背景音乐
- 风格: 现代科技感、轻快节奏
- 音量: 保持在-20dB左右，不影响旁白
- 推荐: 无版权的科技类背景音乐

#### 旁白要求
- 语速: 中等偏快，清晰易懂
- 语调: 专业、自信、友好
- 音质: 清晰无杂音

## 制作时间规划

1. **策划阶段** (2-3天): 脚本完善、素材收集
2. **录制阶段** (3-5天): 界面录屏、功能演示
3. **制作阶段** (5-10天): 视频剪辑、特效制作
4. **优化阶段** (2-3天): 音频处理、最终调色
5. **发布阶段** (1天): 多平台上传、推广
